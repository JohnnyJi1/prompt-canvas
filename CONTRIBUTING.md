# Contributing

Thanks for helping improve Prompt Canvas.

## Principles

- Keep it local-first.
- Avoid adding dependencies unless they unlock clear value.
- Preserve the single-file, easy-to-run experience where practical.
- Prefer small, focused changes.

## Development

Run a local server:

```bash
python3 -m http.server 8765
```

Open:

```text
http://localhost:8765/index.html
```

## Pull Requests

Good pull requests usually include:

- a clear before/after description
- screenshots or screen recordings for UI changes
- notes about LocalStorage migration if data structures change
- manual test steps

## Ideas Welcome

Useful areas:

- prompt templates
- variable workflows
- canvas interactions
- accessibility
- import/export formats
- keyboard shortcuts
